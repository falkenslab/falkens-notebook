Construye el proyecto, inicia el servidor local y abre el navegador.

Ejecuta los siguientes pasos en orden:

1. Ejecuta `npm run build` para compilar el sitio estático. Si hay errores de compilación, muéstralos y detente.
2. Ejecuta `npm run serve` en segundo plano (background) para servir el sitio construido.
3. Abre el navegador en `http://localhost:3000/falkens-notebook`.
