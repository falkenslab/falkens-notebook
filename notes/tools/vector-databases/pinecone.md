# Pinecone

Base de datos vectorial como servicio gestionado (serverless). Escala automáticamente, incluyendo a cero cuando no hay tráfico. Latencia sub-100ms en datasets de miles de millones de vectores sin necesidad de gestión de infraestructura.

## Puntos fuertes

- Serverless: escala automática, sin gestión de nodos ni índices
- Latencia sub-100ms a escala de miles de millones de vectores
- Búsqueda híbrida (vectorial + sparse)
- Namespaces para multitenancy
- SDKs en Python, JavaScript, Go, Java

**Cuándo usarlo:** organizaciones que priorizan cero operaciones sobre el coste; proyectos cloud-native que no quieren gestionar infraestructura.
